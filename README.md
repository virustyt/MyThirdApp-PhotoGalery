# MyThirdApp-PhotoGalery

An iOS application that shows photos from BGSoft server.

## What I Learned
 - MVVM
 - router pattern
 - native networking
 - resizing UIImages
 - UICollectionViews:
   - paging
   - parallax effect
   - infinite scrolling in both directions
   - auto-scrolling when the user is "inactive" for more than 5 seconds
